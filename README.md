# Student Record Management System

## Overview
A simple command-line Student Record Management System built with Python.

This application allows users to:

- Add students
- View students
- Search for students
- Delete students (coming soon)
- Exit the application

## Technologies Used

- Python
- Lists
- Functions
- Conditional Statements
- Loops

## Features

### Add Student
Users can add a student's name to the system.

### View Students
Displays all students currently stored in memory.

### Search Student
Allows users to search for a student by name.

### Delete Student
Feature planned for future implementation.
